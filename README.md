## 📌 Project Description

This project is a **C# Console Cybersecurity Awareness Chatbot** designed to educate users about common online threats and safe internet practices.

The chatbot interacts with users, provides cybersecurity tips, and helps raise awareness about topics such as phishing, malware, password security, and safe browsing.

---

## ⚙️ Features

* 🎤 Voice greeting using a `.wav` audio file
* 💬 Interactive chatbot with menu system
* 👤 User name personalization
* 🎨 Styled console interface with colors and ASCII art
* ⌨️ Typing animation effect for responses
* 🔐 Cybersecurity awareness information
* 🧱 Structured using multiple classes (OOP)

---

## 🛠️ Technologies Used

* C#
* .NET Console Application
* Visual Studio Community

---

## 🧱 Project Structure

* `Program.cs` → Main entry point of the application
* `Chatbot.cs` → Handles chatbot logic and user interaction
* `User.cs` → Stores user information
* `AudioPlayer.cs` → Plays the voice greeting

---

## ▶️ How to Run the Project

1. Open the project in **Visual Studio**
2. Build the solution
3. Run the program using **F5**
4. Enter your name and interact with the chatbot

---

## 🎯 Purpose of the Project

The purpose of this chatbot is to promote **cybersecurity awareness** by educating users on how to stay safe online and avoid common cyber threats.
